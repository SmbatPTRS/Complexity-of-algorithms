# Complexity of Algorithms

Structured study notes on algorithm complexity and asymptotic analysis.

This repository documents a progressive understanding of how algorithms are analyzed, counted, and compared using mathematical tools.

---

## Topics Covered

- Algorithm intuition and cost models
- Counting operations
- Iterative algorithm complexity
- Classical sums and integral comparisons
- Asymptotic notation (O, Ω, Θ)
- Algebra of asymptotics
- Expected complexity and random processes

---

## Learning Approach

The notes are organized in phases.  
Each phase builds intuition first, then introduces formal mathematical reasoning.
